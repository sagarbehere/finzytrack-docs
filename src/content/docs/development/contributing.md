Thank you for your interest in contributing to FinzyTrack!

## Contributing to Documentation

This documentation site is built with [Starlight](https://starlight.astro.build/), an Astro-powered documentation framework.

### Setup

```bash
git clone https://github.com/finzytrack/finzytrack-docs.git
cd finzytrack-docs
npm install
```

### Local Preview

```bash
npm run dev
```

This starts a local server with live reload.

### Making Changes

1. Fork the repository
2. Create a branch for your changes
3. Edit or add Markdown files in `src/content/docs/`
4. Preview locally with `npm run dev`
5. Submit a pull request

## Contributing to FinzyTrack

For contributions to the main application, see the [FinzyTrack repository](https://github.com/finzytrack/finzytrack).
