The Import view lets you bring financial data from external sources into your Beancount ledger.

## Supported Formats

- **OFX** — Open Financial Exchange (commonly exported by banks)
- **CSV** — Comma-separated values with configurable column mapping

## Import Workflow

1. Select the file to import
2. Choose or create an import profile
3. Review parsed transactions
4. Apply import rules for automatic categorization
5. Confirm and save to your ledger

## Import Rules

Rules automatically categorize transactions during import. For the full rule syntax and options, see the [Import Rules](/reference/import-rules/) reference.
