FinzyTrack helps you track, analyze, and understand your personal finances using the power of [Beancount](https://beancount.github.io/) double-entry bookkeeping with a modern web interface.

## Features

- **Double-entry bookkeeping** — powered by Beancount, the gold standard for plain-text accounting
- **Modern dashboard** — customizable widgets and charts for visualizing your finances
- **Data import** — import transactions from bank statements (OFX, CSV) with configurable rules
- **Smart import rules** — define rules to automatically categorize and transform imported transactions
- **Desktop app** — runs as a standalone desktop application
