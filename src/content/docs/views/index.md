FinzyTrack's sidebar provides access to the main views of the application. Each view focuses on a different aspect of your financial data.

| View | Purpose |
| ---- | ------- |
| [Dashboards](/views/dashboards/) | Customizable widgets and charts for visualizing your finances |
| [Import](/views/import/) | Import transactions from bank statements |
| [Transactions](/views/transactions/) | Browse and search your transaction history |
| [Accounts](/views/accounts/) | View your account hierarchy and balances |
| [Query](/views/query/) | Run custom queries against your financial data |
| [Analyze](/views/analyze/) | Advanced analysis and reporting |
