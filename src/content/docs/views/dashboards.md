The Dashboards view is Finzytrack's primary interface for visualizing your financial data at a glance.

## Overview

Dashboards are composed of **widgets** — configurable components that display charts, tables, and key metrics.

## Widget Types

- **KPI** — key performance indicators (single numeric values)
- **Charts** — bar, line, pie, treemap, and more
- **Tables** — tabular data with sorting and filtering

## Customization

Dashboards and widgets are defined using JSON recipe files. See the [Dashboard Recipes](/reference/dashboard-recipes/) reference for full details on creating and customizing dashboards.
